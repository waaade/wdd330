Week 09 Notes

The Window Object

  • Dialogs: alert, confirm, prompt
  • window.location.href = this can actually redirect the page
  • window.history.go(-1) Go back one page in browser history
  • A lot of things in this chapter you really shouldn't use it seems, which is why I'm not making notes about them
  • Cookies are in many ways being replaced by localStorage, but you use the document.cookie property to assign values (they are like regular variables)
  • setInterval: you can have it do something every x milliseconds. setTimeout: have it wait x milliseconds and then do something
  • javascript can do animations but the css method is generally better

HTML5 APIs

  • Data-: lets you attach a bunch of information to an html element
  • navigator.geolocation.getCurrentPosition(callback) Gets your location (literally physical location)
  • Web workers: lets stuff happen in the background. Give them a javascript file
  • Media: tags for audio and video work the same as img. But they have a lot of properties you can control.
